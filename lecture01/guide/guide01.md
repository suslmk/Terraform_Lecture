# 실습 1

## 1.0 준비
terraform 설치
```
brew tap hashicorp/tap
brew install hashicorp/tap/terraform
```
> [설치경로이동(Terraform)](https://developer.hashicorp.com/terraform/downloads?product_intent=terraform)


## 1.1 기본명령어
설치된 terraform 버전확인
```
terraform version
```

help옵션을 통한 사용법 확인
```
terraform --help
```
